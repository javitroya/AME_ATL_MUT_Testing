In order to execute the T2M Transformation:

Go to src->ATL_File2Model. 
Right click on ATLFile2Model.java and select Run As -> Run Configurations
Select Java Application.
In the Arguments tab, write the path of the transformation that you want to inject into a model, as well as the
path where you want the model to be created. For instance:

Transformations/Public2Private.atl
Transformations/Public2Private.xmi
